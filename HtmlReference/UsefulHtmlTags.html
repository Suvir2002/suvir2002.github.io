Most Used Tags

This section includes the tags that I use the most. These are tags that I use in almost every website that I create. Compare this to your most used tags and see if there is anything in common.

Tag Description
<!DOCTYPE html> Defines the document type as HTML
<title> Defines the document's body
<head> Defines information about the document
<body> Defines the document's body
<br> This inserts a single line break
<a> Defines a hyperlink
<img> Defines an image
<!--...--> Defines a comment
<ul> Defines an unordered list
<li> Defines a list item
<div> Defines a division or a section in an HTML document
<h1> to <h6> <h1> defines the most important heading and <h6> defines the least important heading.
<p> Defines a paragraph
<table> Defines a table
<tbody> Groups the body content in a table
<td> Defines a cell in a table
<tfoot> Groups the footer content in a table
<th> Defines a header cell in a table
<thead> Groups the header content in a table
<span> Used to group inline-elements in a document
<footer> Defines a footer for a document or section
<center> Defines centered text
